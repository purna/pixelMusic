üéØ How to Use

  1. Open the Enhanced Editor: Click the button below to open the enhanced node editor
  2. Wait for Audio Loading: The editor loads Strudel and Tone.js from CDN - this may take a few seconds
  3. Test Audio: Click "Test Audio" to verify the audio engine is working
  4. Generate Patterns: Use the pattern generator or click on example patterns
  5. Customize Instruments: Click on instrument nodes to change the instrument type using the dropdown
  6. Test Sounds: Click the play button on individual nodes or use "Play All"
  7. Create Connections: Drag from output to input ports to connect nodes
  8. Edit Properties: Click on editable properties to modify node behavior
